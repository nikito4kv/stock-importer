# Benchmark Fixtures

Фиксированные входные файлы для baseline и повторяемых benchmark-замеров:

- `small-3.docx` — 3 paragraph entries
- `medium-15.docx` — 15 paragraph entries
- `large-40.docx` — 40 paragraph entries

Источник: `test.docx` из корня репозитория, header сохранён, далее взяты первые N
абзацев без изменений.
