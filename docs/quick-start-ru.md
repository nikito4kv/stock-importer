# Быстрый старт Vid Img Downloader

## Вариант 1: самый простой первый запуск со Storyblocks

1. Запустите приложение.
2. Откройте вкладку `Ключи API`.
3. Вставьте и сохраните `Gemini API key`.
4. Если нужны бесплатные картинки, сохраните `Pexels API key` и/или `Pixabay API key`.
5. Откройте вкладку `Сессия`.
6. Нажмите `Войти в браузере`.
7. Войдите в Storyblocks в открывшемся `Chrome` или `Edge`.
8. Не закрывайте это окно.
9. Вернитесь в приложение и нажмите `Проверить сессию`.
10. Дождитесь статуса `Готово`.
11. Перейдите на вкладку `Главная`.
12. В поле `Файл сценария` нажмите `Обзор` и выберите `.docx`.
13. В поле `Папка вывода` нажмите `Обзор` и выберите папку для результатов.
14. В поле `Режим проекта` выберите `Только видео Storyblocks`.
15. Нажмите `Открыть сценарий`.
16. Проверьте блок `Перед запуском`.
17. Нажмите `Старт`.

Готово: прогресс будет виден в блоках `Что сейчас происходит` и `Журнал событий`.

## Вариант 2: работа без Storyblocks

1. Запустите приложение.
2. Откройте `Ключи API`.
3. Сохраните `Gemini API key`.
4. При необходимости сохраните ключи `Pexels` и `Pixabay`.
5. Перейдите на вкладку `Эксперт`.
6. Включите нужные бесплатные источники: `Pexels`, `Pixabay`, `Openverse`, `Wikimedia Commons`.
7. Вернитесь на вкладку `Главная`.
8. Выберите `.docx`-сценарий.
9. Выберите папку вывода.
10. В поле `Режим проекта` выберите `Только бесплатные изображения`.
11. Нажмите `Открыть сценарий`.
12. Нажмите `Старт`.

## Где брать ключи

- `Gemini`: `https://ai.google.dev/gemini-api/docs/api-key`
- `Pexels`: `https://www.pexels.com/api/`
- `Pixabay`: `https://pixabay.com/api/docs/`

`Openverse`, `Wikimedia Commons` и `Bing` в текущем интерфейсе отдельного API key не требуют.

## Что означают главные кнопки

- `Открыть сценарий` - импортирует `.docx` в проект.
- `Старт` - запускает новый run.
- `Продолжить` - продолжает paused run.
- `Пауза` - ставит run на паузу после текущего абзаца.
- `Остановить` - отменяет run после текущего абзаца.
- `Повторить ошибки` - делает новый run только по ошибочным абзацам.
- `Проверить сессию` - проверяет, готов ли Storyblocks.

## Что означают главные режимы

- `Только видео Storyblocks` - только видео, нужен Storyblocks.
- `Видео Storyblocks + изображения Storyblocks` - и видео, и картинки из Storyblocks.
- `Видео Storyblocks + бесплатные изображения` - видео из Storyblocks, картинки из free sources.
- `Изображения Storyblocks + бесплатные изображения` - только картинки, Storyblocks плюс fallback.
- `Только бесплатные изображения` - режим без Storyblocks.

## Где смотреть результат

Файлы текущего запуска обычно попадают сюда:

- `<папка вывода>/<run_id>/downloads/videos/`
- `<папка вывода>/<run_id>/downloads/images/`

## Если что-то не работает

- Проверьте, что сценарий `.docx` действительно нумерованный.
- Для режимов со Storyblocks сначала добейтесь статуса `Готово` во вкладке `Сессия`.
- Для free-image режимов включите хотя бы один бесплатный провайдер во вкладке `Эксперт`.
- Если run завершился частично, откройте `История`, выберите запуск и нажмите `Повторить ошибки`.

Подробная инструкция находится в `docs/user-manual-ru.md`.
